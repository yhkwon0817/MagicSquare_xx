## 1. 마지막으로 틀렸을 때

### Q1 ✅ Mom Test 준수

마지막으로 4×4 마방진(빈칸 2개, 합 34)을 **틀렸다고 느낀** 게 언제였고, 그때 **어떤 줄·대각**이 안 맞았나요?

### Q2 ❌ Mom Test 위반

> *「대각선을 빼먹으면 앱이 알려주면 좋겠어요?」*

| 항목 | 내용 |
|------|------|
| **왜 나쁜지** | 미래·솔루션 가정. “좋겠다”는 의견만 나오고 과거 행동이 안 나옴. |
| **✅ 고친 버전** | 그때 **대각선을 빼먹었다는 걸 언제·어떻게** 알게 됐고, 그 전까지 **몇 분** 더 시도했나요? |

---

## 2. 검증/확인 비용

### Q3 ✅ Mom Test 준수

맞다고 확신하기 전에 **행·열·대각·중복·1~16**을 확인할 때, **몇 분** 걸렸고 **손으로 더한 횟수**는 몇 번이었나요?

### Q4 ❌ Mom Test 위반

> *「자동 검증 기능이 있으면 20분 안 날릴 수 있을 것 같아요?」*

| 항목 | 내용 |
|------|------|
| **왜 나쁜지** | 가설·기능 제안. 실제 행동이 아닌 동의만 유도함. |
| **✅ 고친 버전** | 20분 날린 그날, **확인을 안 한 항목**이 뭐였고, 나중에 **전부 다시 더하는 데** 몇 분 걸렸나요? |

---

## 3. 포기·중단 순간

### Q5 ✅ Mom Test 준수

4×4 마방진을 풀다 **그만둔 적**이 있나요? 있다면 **가장 최근** 한 번, **어디까지** 했고 **뭐 때문에** 멈췄나요?

### Q6 ❌ Mom Test 위반

> *「막히면 힌트 버튼이 있으면 포기 안 할 것 같아요?」*

| 항목 | 내용 |
|------|------|
| **왜 나쁜지** | “~할 것 같다”는 예측. 제품 아이디어에 답하게 만듦. |
| **✅ 고친 버전** | 막혔을 때 **실제로 한 일**이 뭐였나요 — 숫자만 바꿨나요, 표를 지웠나요, 아니면 **과제를 안 냈나요**? |

---

## 4. 같은 문제를 다시 풀 때

### Q7 ✅ Mom Test 준수

같은 격자(빈칸 2개)를 **두 번째** 풀 때, **첫 번째와 뭐가 달랐나요** — 확인 순서, 걸린 시간, 실수 여부?

### Q8 ❌ Mom Test 위반

> *「다시 풀 때 체크리스트 UI가 있으면 편할까요?」*

| 항목 | 내용 |
|------|------|
| **왜 나쁜지** | UI·기능 선호를 묻는 Mom Test 금지 패턴. |
| **✅ 고친 버전** | 두 번째엔 **제출 전에** 행·열·대각·중복·범위 중 **무엇부터** 확인했고, **몇 분** 만에 끝냈나요? |

## 5. 다른 사람에게 설명/공유할 때

### Q9 ✅ Mom Test 준수

4×4 마방진을 **친구·조교·동료**에게 설명하려던 **가장 최근** 때, **어디서 막혔고** 상대가 **뭐라고** 했나요?

### Q10 ❌ Mom Test 위반

> *「공유용으로 ‘맞는지 한눈에 보이는’ 화면이 있으면 설명이 쉬울까요?」*

| 항목 | 내용 |
|------|------|
| **왜 나쁜지** | 솔루션을 먼저 깔고 동의를 구하는 질문. |
| **✅ 고친 버전** | 그때 상대에게 **뭘 보여줬고**(종이·코드·사진), **어느 줄·대각**을 가리키며 설명했나요? |
